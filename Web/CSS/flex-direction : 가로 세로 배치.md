# 🎃 flex direction 🎃
flex-direction을 통해 세로로 배치를 할지 가로로 배치할지를 정해줄 수 있다.

## 👉 가로 : row
```css
flex-direction: row; //가로
```
```css
display: flex;
```
## 👉 세로 : column
```css
flex-direction: column; //세로
```
```css
  display: flex ;
 /*위에서 아래로 수직 배치*/
  flex-direction: column;
  
  /*중앙정렬*/
  justify-content: center;
```
