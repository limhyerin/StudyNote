# 🎃 데이터 타입의 종류 🎃
자바스크립트에서 데이터 값의 타입은 기본형(Primitive Type)과 참조형(Reference Type)으로 구분된다. 기본형과 참조형의 구분 기준은 값의 저장 방식과 불변성 여부이다.

## 💡 [기본형과 참조형의 구분 기준]
### 1. 복제의 방식
(1) 기본형 : 값이 담긴 주소 값을 바로 복제 <br/>
(2) 참조형 : 값이 담긴 주소 값들로 이루어진 묶음들을 가리키는 주소 값을 복제 <br/>

<br/>

### 2. 불변성의 여부
(1) 기본형 : 불변성을 띔 <br/>
(2) 참조형 : 불변성을 띄지 않음 <br/>
