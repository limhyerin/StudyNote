# 🌻env파일에서 변수 값을 가져올때🌻
```
${process.env.REACT_APP_SERVER_URL}
```

## 🌼기존 코드🌼
```js
const fetchTodos = async () => {
    const { data } = await axios.get("http://localhost:4000/todos");
    setTodos(data);
  }
```

## 🌼env파일의 변수 사용🌼
```js
const fetchTodos = async () => {
    const { data } = await axios.get(`${process.env.REACT_APP_SERVER_URL}/todos`);
    setTodos(data);
  }
```
