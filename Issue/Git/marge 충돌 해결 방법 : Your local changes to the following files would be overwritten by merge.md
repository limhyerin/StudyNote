# 🚨error code🚨
Your local changes to the following files would be overwritten by merge
 
<br/>

## 1. staging 영역에 있는 파일의 변경사항 스택에 넣기
```bash
git stash
``` 

## 2. master나 main 혹은 dev 브랜치 같이 같이 작업하는 브랜치에서 pull을 해와서 내 로컬 브랜치로 변경사항 저장
```bash
git pull origin dev
```

## 3. 변경사항을 적용하고 스택에서제거
```bash
git stash pop
```
