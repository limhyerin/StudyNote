## 04 파이프, grep

: test.txt 파일에서 ~이 있는 행만 출력해라

## 05 시스템 종료 및 재시작

### - 시스템 즉시 종료
```
shutdown -h now
halt
init 0
```
    
### - 즉시 재시작
```
shutdown -r now
reboot
init 6
```
